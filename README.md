# 数学题生成器

一个简单的加减法题目生成器. 供打印使用(或口算).

一次生成60题, 正好一张A4纸.

![](./public/thumb.png)


## 功能

- [x] 加法
- [x] 减法
- [ ] 乘法
- [ ] 除法
- [x] 加法进位
- [x] 减法退位
- [x] 直接显示答案
- [x] 扫码显示答案
- [x] 打赏
- [x] 单题刷新


## 技术栈

- vite
- 伪ts
- svelte
- unocss
