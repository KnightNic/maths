# 数学题生成器

简单的一个加减法生成器.

一次生成60题, 正好一张A4纸.


## 功能

- [x] 加法
- [x] 减法
- [x] 加减法 
- [ ] 加法进位
- [ ] 减法退位
- [ ] 两位与一位运算
- [ ] 两位与两位运算
